<h3><b>Project Description</b></h3>
<p>Cruise booking website where user can able to search according to its requirement and after that it will navigate to result page where all the results are showed and then user have the option to book from the options available on result page. It also has feature where user can explore different places to plan their trip. Some additional features include of careers portal where user can apply for the job, explore famous hotels and get necessary accessories for the trip from the accessories store.</p>
